# Assessment-as-a-Service (B2B) Go-To-Market Plan

## Objective
Launch a revenue-focused B2B assessment product leveraging the existing assessment module, AI interview flow, and self-evolution system. This plan prioritizes fast time-to-revenue and measurable hiring outcomes.

## Ideal Customer Profile (ICP)
- Small-to-mid tech companies (10–200 employees)
- Hiring managers, CTOs, or HR leads hiring engineers
- Pain points: inconsistent screening, slow hiring, weak signal-to-noise

## Core Product Offer
### MVP Scope (Weeks 1–2)
- **HR Dashboard**: view candidates, invite links, status, scores
- **Assessment Flow**: coding test + AI interview
- **Candidate Report** (PDF/HTML):
  - summary score
  - strengths/risks
  - interview transcript highlights
  - recommended next step

### Differentiators
- AI interview tailoring per role
- Self-evolution pipeline improving test quality over time
- Local model option for data-sensitive orgs

## Pricing & Packaging
### Subscription + Usage
- **Starter**: $99/mo → up to 10 assessments
- **Growth**: $399/mo → up to 50 assessments
- **Enterprise**: $999+/mo → custom volume + SSO + SLA

### Add-ons
- Role-specific interview packs
- Custom rubric + scoring rubric
- Branded candidate experience

## Distribution Strategy
### Channels
- LinkedIn outbound to hiring managers/CTOs
- Direct outreach to local tech communities
- Partnerships with bootcamps or training schools

### Outreach Script (Short)
"We help reduce hiring time by running a structured AI-based screening + coding assessment. Want to pilot with 5 candidates this week?"

## 6-Week Execution Plan
### Weeks 1–2: MVP Build
- Build HR dashboard and candidate reporting view
- Add invite workflow + tracking
- Instrument analytics (completion rate, drop-off, avg score)

### Weeks 3–4: Monetization
- Add Stripe payment + usage limits
- Deploy pricing page + FAQ
- Record demo video

### Weeks 5–6: Sales & Feedback
- Run 10–20 outbound outreach sequences
- Secure 3–5 pilots
- Collect feedback and iterate on report quality

## Success Metrics
- 3+ paying pilots within 6 weeks
- 20+ completed assessments
- >60% completion rate
- >30% follow-up conversion
